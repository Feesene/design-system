## `Login`

Fazer login no NPM:

> ```bash
> npm login --registry https://npm.pkg.github.com/
> ```

## `Instalar`

Instalação da biblioteca:

> ```bash
> npm install design-system
> ```

## `Atualizar`

> ```bash
> npm install design-system@latest
> ```
