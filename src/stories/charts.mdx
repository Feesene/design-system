### <i>NEXT JS</i>

Para integrar o componente de gráfico de área em uma aplicação Next.js, utilize a função dynamic para carregar o componente de forma assíncrona. Isso é especialmente útil quando o componente não precisa ser renderizado no lado do servidor (SSR).

<Source
  code={`
import dynamic from "next/dynamic";

const AreaChart = dynamic(() => import("@design-system/charts").then((value) => value.AreaChart), { ssr: false });

`}
  language="tsx"
  type="auto"
  dark
/>

`ssr: false` desativa o carregamento do componente no lado do servidor, tornando-o exclusivamente client-side. Isso é útil quando o componente depende de recursos que não estão disponíveis no servidor.
